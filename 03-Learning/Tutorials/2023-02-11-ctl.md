# 如何打造炫酷又实用的终端（一）ohmyzsh的配置

大家好，我是xxx，今天给大家介绍我是如何打造炫酷又实用的终端。

首先，安装zsh。这一步各大包管理器都自带了相关的包，安装即可，不再赘述。

第二步，安装omyzsh。按照官网（https://ohmyz.sh/）给出的脚本安装即可。

如果遇到网络问题，可以开代代理，或者使用清华大学开源软件镜像站（https://mirrors.tuna.tsinghua.edu.cn/help/ohmyzsh.git/）提供的镜像和脚本。

第三步，安装zsh-autosuggestions和zsh-syntax-highlighting。依照两者官网（https://github.com/zsh-users/zsh-autosuggestions/blob/master/INSTALL.md 和 https://github.com/zsh-users/zsh-syntax-highlighting/blob/master/INSTALL.md）给出的步骤安装。

第四步，配置插件。添加z，fzf这样的插件。
