# LocalSend

大家好，我是xxx。今天呢和大伙聊一聊一个经常遇到的问题，如何在不同设备之间传递文件。

根据我对身边人的观察，大部分人还是会用微信qq来传递文件，比如把工作文档发给同事，又或者把手机照片发到电脑上。

应该说，这种办法对大多数人来说足够用了。

但很多小伙伴还是会遇到满足不了的时候，比如传递一些大文件，或者文档内容需要保密，不能上传到微信qq这样的远端服务器上。

这时肯定会有小伙伴就会想到给手机接上数据线连到电脑上来传递。这也是一种办法。

但有时没有数据线，或者是需要在两台手机之间两台电脑之间传递又该怎么办呢。

这就需要我们今天给大家介绍的这款软件——LocalSend。

LocalSend是一款可以在局域网内部传递文件的应用，支持的平台包括安卓，windows，macOS。
