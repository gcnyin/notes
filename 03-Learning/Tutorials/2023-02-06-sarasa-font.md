# 更纱黑体

大家好，我是xxx，今天给大家推荐一款编程用的字体：Sarasa。

Sarasa 是一款由中国人创作的字体，作者是 Belleve，中国人，在微软工作。从他的知乎个人主页上看，目前应该居住在美国西雅图。

Sarasa 是作者结合了思源黑体与作者的另一个开源字体 Iosevka 后的成果。

Sarasa 和 Iosevka 的关系呢，有点像奶茶和茶的关系。Iosevka 不包含中文日文韩文，Sarasa 借助思源黑体将这部分补全了。这里必须提醒下，由于编程时主要用到的是英文和各种符号，我们接下来在讲 Sarasa 时，很多优点其实都是 Iosevka 的。

Sarasa有这样几个优点。

一、等宽。编程开发时用的字体一般来说都是等宽的，非常方便开发人员较对代码的缩进和空格。

二、字符不容易混淆。比如0和O，大写的I和小写的l。

三、字重非常丰富。表现力更强。

四、各类符号齐全。一些不常用的字符也涵盖了，避免一些特殊情况。
