# Windows 11 必装软件

下载助手：http://gh-proxy.com/

## 通用

- ClashVergeRev. `winget install ClashVergeRev.ClashVergeRev`
- TrafficMonitor. `winget install zhongyang219.TrafficMonitor`
- GeekUninstaller. `winget install GeekUninstaller.GeekUninstaller`
- 7zip. `winget install 7zip.7zip`
- 1Password. `winget install AgileBits.1Password`
- Firefox. `winget install Mozilla.Firefox.zh-CN`

## 游戏

- Steam. https://store.steampowered.com/about/

## 开发工具

- Git. `winget install Git.Git`
- Visual Studio Code. `winget install Microsoft.VisualStudioCode`
- OpenJDK. `winget install Microsoft.OpenJDK.21`
- NodeJS. `winget install OpenJS.NodeJS`
- Python. `winget install Python.Python.3.14`
- Rustup. `winget install Rustlang.Rustup`
- Golang. `winget install GoLang.Go`
- Dotnet. `winget install Microsoft.DotNet.SDK.10`


## AI工具

- Codex. `npm i -g @openai/codex`
- Codex App. https://chatgpt.com/codex
- Qwen Code. `npm i -g @qwen-code/qwen-code`
