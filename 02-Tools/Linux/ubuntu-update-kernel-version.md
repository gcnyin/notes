# 升级内核版本

1.  下载debs，手动dpkg
2.  使用网上流传的pimlie/ubuntu-mainline-kernel.sh 脚本
3.  apt install linux-image-VERSION
4.  使用Ukuu或UKTools这样的三方工具
5.  添加ppa:cappelikan/ppa，apt install mainline
6.  安装oem版本。apt install linux-oem-20.04b
7.  安装https://ubuntu.com/kernel/variants 提供的版本，包括generic-hwe
