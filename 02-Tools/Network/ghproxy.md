# gh-proxy

https://gh-proxy.com/

github国内下载加速，在链接`https://gh-proxy.org`后加原始链接即可，git项目clone也一样。

原始链接比如

- https://github.com/clash-verge-rev/clash-verge-rev/releases/download/v2.4.6/Clash.Verge_2.4.6_x64-setup.exe

转换后加上前缀就是

- https://gh-proxy.org/https://github.com/clash-verge-rev/clash-verge-rev/releases/download/v2.4.6/Clash.Verge_2.4.6_x64-setup.exe
