# 查看定位隐藏分

设置中勾选，启动控制台，打开控制台。

控制台先输入 developer 1 ，然后再输入 dota_game_account_client_debug 就能看了。

handicap 字面意思是“障碍”，role_hanndicaps 实际意思是“位置劣势”，越接近1表示这个位置玩的越不好，而0表示发挥正常呢？

```
role_handicaps {
  role: 1
  handicap: 0
}
role_handicaps {
  role: 4
  handicap: 0.05
}
role_handicaps {
  role: 2
  handicap: 0.05
}
role_handicaps {
  role: 8
  handicap: 0.05
}
role_handicaps {
  role: 16
  handicap: 0.05
}
```

# 教学博主推荐（转自nga）

别看那些陪玩UP主（白马之类），楼上推荐的什么阿光也是废话一堆，久远Vortex不知道在讲什么抽象理论。我说得难听点，有些人发视频目的是让你觉得他很厉害搞噱头，而不是让你学到什么。

推荐几个UP

1. 陈天宇_Cty（很久没更新了，以前的几个视频看看思路）、DOTA2肉夹馍（刷钱路线，和部分英雄教程比如米波斯温，其他视频随便看看就行）、youc（主要是看看套路，我无聊会看他直播下饭）
2. Ori曾焦阳（复盘）、Quinn（有翻译BV1FzrdBbEwq）
3. 没几个优质的，像多多利亚禁忌可乐什么的营养不多，随便找点凑合看看，对线不爆出团队装就行了
4. lovelesszzx
5. DY小课堂（找切片）

- 油管KheZu：细节分析（有翻译比如BV1Mha1zrET1、BV1Tj1WBfEXn）
- dota2大傻：高分视角
- DiongDiong的白金小教室：英雄机制

重点就是找职业选手学，看录像看直播，那些1W分以下的我看几分钟就皱眉头了。想上分只需要学会：公式出装+把握强势期+对线细节，出装加点去dota2protracker找参考，其他都是看视频自己打积累经验，学会自己复盘，找别人复盘不是好办法，不如挂个教练语音嘴对嘴教你玩。
